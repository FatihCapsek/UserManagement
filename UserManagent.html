Save a New User

To register a new user, you need to be a Super Admin

For save a new user first you have to click New User button. After button click you have to fill blank boxes with user information. Registering this user as Guest, Admin or Super Admin You need to select the role from user Roles. If you actived the Enabled button user will be able to log in after saved.

Once the informations has been entered, a new user can be created by pressing the Save User button

ProTip: You can hide disabled users by use Hide Disabled User button.

Sorting
